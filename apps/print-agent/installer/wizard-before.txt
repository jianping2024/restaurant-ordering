Before you install
==================

SmartScreen (unsigned build)
  Windows may show "Unknown publisher". Click More info, then Run anyway.
  Or: right-click the installer → Properties → check Unblock → OK.

USB printers (UNYKA UK56009, etc.)
  Install the vendor driver first so the printer appears in Windows Settings → Printers.
  Driver info: https://unykach.com/

Portable zip vs this Setup
  This installer can add a startup shortcut (optional task) and uninstall cleanly.
  The portable zip does NOT configure sign-in autostart — use Setup for daily POS PCs.
