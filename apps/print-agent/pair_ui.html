Pair Mesa Print Agent

Generate a 6-digit code in Mesa → Settings → Print assistant, then connect here. No command line.

  1. Sign in to Mesa → Settings → Print assistant → Generate code
  2. Enter your Mesa URL below (check if pre-filled)
  3. Enter the 6-digit code and connect
