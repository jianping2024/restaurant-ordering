③ 试打确认

保存映射后，请对至少一个档口试打。代理已把数据送到打印机，但您仍需目视确认纸上出现「打印测试」。

纸上是否已出现「打印测试」字样？

排障建议
  1. USB：线是否接在这台收银电脑；驱动是否已装；Windows「设置 → 打印机」里是否能看到设备。
  2. 队列名：所选打印机是否与实体机一致（勿选 PDF / 传真 / 旧队列）。
  3. 走纸：按打印机 FEED 键看是否进纸。
  4. 脱机/暂停：Windows 打印队列中是否显示暂停、脱机或错误。

修正后请重新「打印测试条」，确认出纸后再关闭本页。
